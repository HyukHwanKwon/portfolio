# dogShop
강아지 분양 관리자 사이트

### 사용 OS : windows10
### 사용 DB : mysql 
### 사용 언어 : java, jsp, html5, css, jquery, javascript, jstl
### 사용 Tool : eclipse
### 프로젝트 참여인원 : 1명 
### mvc2패턴을 사용하여 개발하였습니다.
